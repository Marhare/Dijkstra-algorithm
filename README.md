
# Dijkstra Pathfinding Implementation

This project is a visual implementation of **Dijkstra's Algorithm**, a fundamental graph search algorithm used to find the shortest path between nodes in a graph.

## Project Goal
The objective was to practice programming logic and data structures by visualizing how the algorithm explores a grid/graph and determines the optimal path based on edge weights.
